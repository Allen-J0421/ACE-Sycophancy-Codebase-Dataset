# Two Pointers Technique

Small Java example showing two ways to solve the pair-sum problem:

- `TwoPointersTechnique.hasPairWithSum(...)` for unsorted input
- `TwoPointersTechnique.hasPairWithSumSorted(...)` for sorted input

## Layout

- `src/two_pointers/TwoPointersTechnique.java` - core implementation
- `src/two_pointers/TwoPointersTechniqueDemo.java` - demo entry point
- `src/two_pointers/TwoPointersTechniqueTest.java` - lightweight test harness
- `src/two_pointers/TwoPointersTechniqueSamples.java` - shared sample data
- `Makefile` - build, test, run, and clean targets

## Commands

- `make` or `make test` - compile and run the test harness
- `make run` - compile and run the demo
- `make clean` - remove generated output
